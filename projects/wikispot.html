WikiSpot

IT academy in slovakia started competition in partnership with Telekom. We started doing project for smart city theme and we named our creation WikiSpot.

What it is?

It is network of raspberry pi's or other servers in p2p connection. Every RPI has data about location it is in and user can acces data on all the servers in the network by connecting to one of the servers using Wi-Fi and our android app.

What did we use

We had four members and each of them took care of one system:

  • Server/client/linux
  • Computer vision
  • Mobile app
  • Webpage

Server

I developed the server. Because it was meant for RPI I also created a image for sd card that will automaticaly configure the server. The servers can comunicate with each other and keep track of which one is online.

It is possible thanks to "heartbeat table" that is shared and merged with every ping between servers. It stores basic information about every server like:

  • Last seen
  • Offline/online
  • Files available on server
  • Location
  • ip addres
